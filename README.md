
# BIZCARDX - A Business card extractor.


This project includes python scripting in which data from digital business card image gets converted into storable relevant data.


## Features

- streamlit interface
-  ocr reader
- data storage
- modifcations to the stored data


## Required packages

- Pandas
- Streamlit
- Easyocr
- SQL connector
- Matplotlib
- Regular expression
